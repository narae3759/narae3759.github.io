# 2. 서버 실행
1. `main.py` 파일을 만든다.
2. (cmd) set FLASK_APP=pybo
3. (cmd) flask run

> ***<font color='red'>Error</font>*** <br>
> ```
> Usage: flask run [OPTIONS]
> Try 'flask run --help' for help.
> ```
> powershell 인지 확인해야 한다. 반드시 cmd창에서 진행해야 한다.

* 실행한 후부터 cmd 창에 log가 나타나게 된다. <br>
그럼 올바르게 서버 접근이 가능한지를 알 수 있다.

# 3. 서버 계속 띄우기
* 위의 방식으로 하게 되면 pycharm 프로그램을 종료할 경우 서버 또한 종료된다.
* cmd창에서 해당 가상환경을 활성화 한다
    ```
    conda activate flask
    ```
* `main.py` 경로를 찾아서 서버를 실행한다.
